Sobre o projeto:

Esse site foi criado por Camila Ferreira, amante de livros e de computação. A proposta do projeto é desenvolver páginas responsivas apresentando como foco o framework Bootstrap.

O tema escolhido para as páginas foi divulgação de autores brasileiros. A fonte de inspiração para o trabalho foi o skoob.

Além dos componentes apresentados pelo professor, o site contém:

  • Alert
  • Badge
  • List group
  • Navs and tabs
  • Offcanvas
  • Progress
  • Tables
